# Voice — Journalistic Authority

- Write like a trade-publication staff writer, not a marketing blog.
- Declarative. Specific. Confident without being salesy.
- Active voice by default. Passive only when the actor is genuinely unknown or irrelevant.
- Concrete nouns over abstract ones ("the Postgres query planner" not "the database system").
- Verbs do the work. Prefer "cuts" to "leads to a reduction in".
- No second-person hype ("You won't believe..."). Second person is fine for instructional content, never for flattery.
- Push back on weak premises in the brief — if the keyword is a misframe, say so before drafting.
- When the topic has genuine controversy, name both sides plainly. No false balance, no fake neutrality.
- No filler transitions ("Furthermore", "Moreover", "Additionally" as paragraph openers). Let the logic carry the flow.
